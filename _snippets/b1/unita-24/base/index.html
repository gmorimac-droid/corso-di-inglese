    Unità 24 – First Conditional (If...)

    🎯 Obiettivo

    Alla fine di questa unità saprai:

    • usare il First Conditional per parlare di situazioni future possibili
    • costruire frasi corrette con if + Present Simple, will + verbo base
    • distinguere tra condizione e risultato
    • evitare gli errori più comuni, soprattutto l’uso scorretto di will dopo if

    🧠 Grammar in context

    Il First Conditional si usa per parlare di una situazione futura reale o possibile e della sua conseguenza.

    Formula base:

    If + Present Simple, will + base form

    • If I study, I will pass the exam.
    • If it rains, we will stay at home.
    • If you call me, I will help you.

    Questa struttura esprime una relazione logica:

    • condizione → cosa deve succedere
    • risultato → cosa succederà come conseguenza

    1. Struttura base

    Formula

    If + subject + Present Simple, subject + will + base form

    • If she arrives early, we will start on time.
    • If they don’t hurry, they will miss the train.
    • If I have time, I will finish the report.

    Ordine alternativo

    Puoi anche invertire l’ordine delle due parti:

    • If I have time, I will call you.
    • I will call you if I have time.

    Quando la frase comincia con if, si usa la virgola.

    2. Forma negativa

    Negativa nella condizione

    • If I don’t study, I won’t pass.
    • If she doesn’t come, we will start without her.

    Negativa nel risultato

    • If it rains, we won’t go out.
    • If you don’t ask, they won’t know.

    Negativa in entrambe le parti

    • If we don’t leave now, we won’t arrive on time.

    3. Forma interrogativa

    Si usa spesso per fare domande su conseguenze future.

    • What will you do if it rains?
    • Will they come if we invite them?
    • What will happen if you miss the bus?
    • If he doesn’t study, will he fail the test?

    4. Quando si usa

    Usa il First Conditional per:

    A. Situazioni future possibili

    • If I see Anna tomorrow, I will tell her.
    • If we go by train, we will arrive earlier.

    B. Avvisi e conseguenze

    • If you touch that, you will break it.
    • If you don’t sleep enough, you will feel tired.

    C. Promesse e decisioni legate a una condizione

    • If you need help, I will stay.
    • If they agree, we will start next week.

    📚 Examples in context

    • If I finish work early, I will go to the gym.
    • If we don’t book now, we will lose the discount.
    • If you study every day, you will improve quickly.
    • If the weather is good, we will have lunch outside.
    • If he doesn’t answer, I will send a message.

    🧩 Time expressions often used

    Nel First Conditional trovi spesso parole come:

    • tomorrow
    • tonight
    • next week
    • later
    • soon
    • in the future
    • this afternoon

    Esempi:

    • If she arrives tomorrow, we will talk.
    • If I have time later, I will call you.

    ⚠️ Regola fondamentale

    Dopo if non si usa will.

    ❌ If I will study, I will pass. ✅ If I study, I will pass.

    ❌ If it will rain, we will stay home. ✅ If it rains, we will stay home.

    Ricorda:

    • if-part = Present Simple
    • main clause = will + verbo base

    🎧 Reading / Dialogue

    Dialogue 1

    A: What will you do if the meeting finishes early? B: If it finishes early, I will go back to the office.

    A: And what if the client has more questions? B: Then we will stay longer.

    Dialogue 2

    A: Will you go out if it rains? B: No, I won’t. If it rains, I will stay home.

    🗣️ Speaking

    Ripeti:

    • If I study, I will pass.
    • If it rains, we will stay home.
    • If you call me, I will answer.
    • If she doesn’t hurry, she will be late.

    Ora completa a voce:

    • If I have free time, I will ...
    • If my friend needs help, I will ...
    • If it is sunny this weekend, I will ...
    • If I don’t feel well, I will ...

    ✍️ Writing

    Scrivi 8 frasi con il First Conditional.

    Usa almeno questi temi:

    • studio
    • lavoro
    • salute
    • famiglia
    • tempo libero
    • viaggi

    Esempio:

    If I finish my tasks early, I will go for a walk.

    🔊 Pronunciation

    Nel parlato naturale:

    • I will → spesso I’ll
    • we will → we’ll
    • he will → he’ll

    Esempi:

    • If I have time, I’ll help you.
    • If they arrive early, we’ll start.

    Allenati a leggere:

    • If you need me, I’ll be here.
    • If it rains, we’ll cancel the picnic.

    ⚠️ Errori comuni (italiani)

    ❌ If I will see her, I will tell her. ✅ If I see her, I will tell her.

    ❌ If he will not study, he will fail. ✅ If he doesn’t study, he will fail.

    ❌ If I study, I pass the exam. ✅ If I study, I will pass the exam.

    ❌ If I will have time, I call you. ✅ If I have time, I will call you.

    🔄 Practice

    Completa con la forma corretta

    1. If I ______ (finish) early, I will come.
    2. If she ______ (not hurry), she will miss the bus.
    3. If we ______ (book) now, we will save money.
    4. If it ______ (rain), we will stay inside.
    5. If they ______ (not study), they will fail.

    Trasforma

    1. I will help you. / you ask me
    2. We will stay home. / it rain
    3. She will be happy. / she pass the exam
    4. They will miss the train. / they not leave now

    🧠 Think & Speak

    Rispondi a voce:

    • What will you do if you are tired tonight?
    • What will you do if you have free time this weekend?
    • What will happen if you don’t study English regularly?
    • What will you buy if you go shopping tomorrow?

    ✅ Self-check

    Controlla se sai:

    • formare frasi con if + Present Simple
    • usare will nel risultato
    • fare domande con il First Conditional
    • evitare will dopo if
    • parlare di conseguenze future possibili

    Se riesci a farlo con sicurezza, la unità è acquisita.
