# Unità 24 – First Conditional (If...)

---

## 🎯 Obiettivo

Alla fine di questa unità sarai in grado di:

- parlare di situazioni future possibili
- usare frasi con *if* in modo corretto
- fare previsioni e dare consigli
- costruire frasi logiche

---

## 🧠 Grammar in context

### Struttura base

👉 **If + present simple, will + verb**

- If I study, I will pass  
- If it rains, I will stay home  

---

## 🔁 Ordine delle frasi

👉 puoi invertire:

- If I study, I will pass  
- I will pass if I study  

---

## 📚 Esempi reali

- If I work hard, I will succeed  
- If you eat too much, you will feel bad  
- If it rains, we will stay home  

---

## 📚 Core vocabulary

### Verbi utili

- study  
- work  
- eat  
- go  
- stay  
- help  

---

### Espressioni utili

- if  
- when  
- maybe  
- probably  

---

## 🎧 Listening / Reading

**Dialogue:**

A: What will you do if it rains?  
B: If it rains, I will stay home.  

A: What will you do if you have time?  
B: I will go to the gym.

---

## 🗣️ Speaking

Ripeti:

- If I study, I will pass  
- If it rains, I will stay home  

---

👉 Ora prova:

- If I ___, I will ___  

---

👉 Fai domande:

- What will you do if ___?  

---

## ✍️ Writing

Scrivi 5 frasi:

👉 usa *if*

Esempio:

> If I study, I will pass the exam.  
> If it rains, I will stay home.  
> If I have time, I will go out.

---

## 🔊 Pronunciation

Attenzione:

- if → /ɪf/  
- will → spesso debole (*I’ll*)  

👉 parlato naturale:

- If I study, I’ll pass  

---

## ⚠️ Errori comuni (italiani)

❌ If I will study, I will pass  
✅ If I study, I will pass  

❌ If it will rain, I will stay home  
✅ If it rains, I will stay home  

❌ If I study, I pass  
✅ If I study, I will pass  

---

## 🧠 Regola fondamentale

👉 dopo *if*:

❌ NO futuro  
✅ sempre present simple  

---

## 🧠 Pattern mentale

👉 pensa così:

- situazione possibile → risultato  

👉 causa → effetto  

---

## 🔁 Confronto reale

- If I study → I will pass  
- If I don’t study → I will fail  

---

## ✅ Review & Practice

Completa:

1. If I ___ (study), I will pass  
2. If it ___ (rain), I will stay home  
3. If I ___ (have) time, I will go out  
4. If you ___ (eat) too much, you will feel bad  

---

## 🔄 Trasforma

Completa:

1. I study → If I ______  
2. I go out → If I ______  
3. I eat → If I ______
