## 🎯 Obiettivo

Alla fine di questa unità sarai in grado di:

- nominare oggetti quotidiani molto comuni
- usare **this / that / these / those**
- distinguere singolare e plurale
- fare domande semplici su oggetti vicini o lontani

---

## 🧠 Grammar in context

### This / That

Si usano per parlare di **un solo oggetto**.

- **this** = questo / questa → vicino
- **that** = quello / quella → lontano

Esempi:

- This is a book.
- That is a chair.

---

### These / Those

Si usano per parlare di **più oggetti**.

- **these** = questi / queste → vicini
- **those** = quelli / quelle → lontani

Esempi:

- These are books.
- Those are chairs.

---

### Schema rapido

| Vicino | Lontano |
|---|---|
| this | that |
| these | those |

---

### Domande utili

- What is this?
- What is that?
- What are these?
- What are those?

Risposte:

- It’s a phone.
- They’re books.

## 🎧 Listening / Reading

**Dialogue:**

A: What is this?  
B: It’s a laptop.

A: Is that your bag?  
B: Yes, it is.

A: What are these?  
B: They’re notebooks.

A: And what are those?  
B: Those are chairs.

## 🔊 Pronunciation

### Il suono **TH**

Parole importanti:

- this
- that
- these
- those

👉 Non dire **dis** o **dat**.

Prova a ripetere lentamente:

- this / these
- that / those

---

## ⚠️ Errori comuni

❌ This are books.  
✅ These are books.

❌ These is a pen.  
✅ This is a pen.  
✅ These are pens.

❌ That are chairs.  
✅ Those are chairs.

❌ This is books.  
✅ These are books.

👉 Controlla sempre:
- singolare o plurale
- vicino o lontano
- **is** o **are**
