# Unità 24 – First Conditional (If...)

---

## 🎯 Lesson Goals

Alla fine di questa unità sarai in grado di:

- usare il **First Conditional** per parlare di conseguenze future realistiche e prevedibili
- distinguere tra una semplice frase sul futuro e una struttura che esprime **condizione + risultato**
- costruire ipotesi pratiche, avvisi, offerte, piani e decisioni legate a circostanze specifiche
- usare con naturalezza schemi molto frequenti nel parlato reale, come **If you need anything, I’ll...**, **If the weather improves, we’ll...**, **If we don’t leave now, we’ll...**
- evitare gli errori tipici degli italofoni, soprattutto **will** dopo **if**, l’uso meccanico del presente al posto del futuro e la punteggiatura incerta

A livello **B1** il First Conditional non serve solo a “fare frasi con if”, ma a ragionare in modo chiaro su conseguenze, scenari probabili, problemi pratici, piani alternativi e risultati attesi.

---

## 🧠 Core idea: a real future condition with a likely consequence

Il **First Conditional** si usa quando immagini una situazione futura **possibile, concreta, plausibile** e ne indichi la conseguenza.

Struttura canonica:

**If + Present Simple, will + base form**

- If we leave now, we’ll catch the train.
- If you don’t revise, you’ll make mistakes.
- If the client agrees, we’ll sign the contract.

Il significato è questo:

- la situazione non è certa al 100%
- ma è realistica
- e il risultato è presentato come una conseguenza logica

Non stai descrivendo un’abitudine generale, e non stai nemmeno immaginando una fantasia improbabile. Stai parlando di un **possibile futuro reale**.

---

## 1. Form and structure

### Basic formula

**If + subject + Present Simple, subject + will + base form**

- If I have time, I’ll call you.
- If she arrives late, we’ll start without her.
- If they don’t answer, we’ll send an email.

### Alternative word order

Puoi invertire le due parti:

- If I have time, I’ll call you.
- I’ll call you if I have time.

La differenza non è grammaticale ma di enfasi:

- mettendo **if-clause** all’inizio, metti in primo piano la condizione
- mettendo prima il risultato, metti in primo piano l’azione principale

### Comma rule

- **If-clause first** → di norma metti la virgola
- **Main clause first** → di norma niente virgola

Corretto:

- If the meeting ends early, we’ll have lunch together.
- We’ll have lunch together if the meeting ends early.

---

## 2. Why Present Simple after *if*?

Questo è il punto che crea più problemi agli italiani.

In italiano diciamo spesso:

- *Se pioverà, resteremo a casa.*

In inglese standard, invece, dopo **if** si usa il **Present Simple**, non **will**.

✅ If it **rains**, we’ll stay at home.
❌ If it **will rain**, we’ll stay at home.

La logica inglese è questa:

- la subordinata con **if** introduce la condizione
- la principale esprime la conseguenza futura

Quindi:

- **if-clause** = present form
- **result clause** = future/result form

---

## 3. Main communicative uses

### A. Plans and practical consequences

- If we leave at six, we’ll avoid traffic.
- If you book today, you’ll get a lower price.
- If I finish this report, I’ll send it tonight.

Qui il First Conditional è molto utile per organizzare azioni concrete.

### B. Warnings and advice with consequences

- If you don’t back up your files, you’ll lose your work.
- If you eat too fast, you’ll feel sick.
- If he keeps interrupting people, they won’t listen to him.

Qui la struttura esprime un rapporto chiaro tra comportamento e risultato.

### C. Offers and support

- If you need help, I’ll stay after class.
- If you want, I’ll check the document for you.
- If there’s a problem, I’ll call the hotel.

Questa funzione è molto comune nel parlato collaborativo e professionale.

### D. Promises and reassurance

- If you tell me the truth, I won’t get angry.
- If everything goes well, I’ll let you know immediately.
- If you can’t come, I’ll explain the situation to the others.

### E. Cause-effect reasoning

- If sales increase, we’ll hire more staff.
- If the weather improves, the event will be more successful.
- If people work remotely more often, city centres may change.

---

## 4. First Conditional vs zero and second conditional

Per usare bene il First Conditional, devi sentirne i confini.

### First Conditional = future possibility

- If it rains tomorrow, we’ll stay home.

Parli di una situazione futura specifica.

### Zero Conditional = general truth / routine consequence

- If it rains, the streets get slippery.

Questa frase non parla di domani: parla di una verità generale.

### Second Conditional = hypothetical / less real

- If it rained tomorrow, we would stay home.

Questa forma suggerisce maggiore distanza o minore probabilità, oppure uno stile più ipotetico.

A livello B1 è importante capire che il **First Conditional** è la forma più naturale per scenari futuri concreti e realistici.

---

## 5. The result clause is not always only *will*

La struttura base è **if + present, will + verb**, ma il risultato può anche contenere altri modali o strutture quando il significato lo richiede.

Esempi naturali:

- If you finish early, **can** you call me?
- If the traffic is bad, we **might** be late.
- If you want to improve, you **must** practise regularly.
- If she arrives before me, tell her to wait.

Per questa unità la struttura principale resta con **will**, ma è utile iniziare a vedere che nel mondo reale la seconda parte può adattarsi al messaggio.

## 6. High-frequency patterns in real English

Questi schemi compaiono continuamente:

### If you need..., I’ll...

- If you need anything, I’ll be in my office.
- If you need more information, I’ll send it later.

### If we don’t..., we’ll...

- If we don’t leave now, we’ll be late.
- If we don’t act quickly, we’ll lose the opportunity.

### If ... happens, I’ll...

- If anything changes, I’ll text you.
- If there’s a problem, I’ll deal with it.

### If the weather / traffic / meeting ...

- If the weather is good, we’ll eat outside.
- If the traffic is heavy, we’ll take the train.
- If the meeting runs late, we’ll cancel lunch.

Imparare questi pattern ti rende subito più sciolto.

### Travel

- If the hotel is too expensive, we’ll look for another one.
- If it stops raining, we’ll go for a walk by the sea.

### Everyday life

- If I get home early, I’ll cook dinner.
- If you don’t charge your phone, it’ll die.
- If they invite us, we’ll probably go.

## 🎧 Reading: practical decision-making

**Situation: planning a weekend trip**

Marta: What shall we do if the weather is bad on Saturday?

Leo: If it rains, we’ll visit the museum instead of going hiking.

Marta: Good idea. And what if the trains are delayed?

Leo: If that happens, we’ll take the earlier one on Sunday.

Marta: Should we book the hotel now?

Leo: Yes. If we book tonight, we’ll probably get a better rate.

### Why this dialogue is useful

Qui il First Conditional serve per:

- fare piani alternativi
- gestire imprevisti
- ragionare in anticipo
- proporre soluzioni concrete

## 🔊 Pronunciation and connected speech

Nel parlato reale **will** spesso si riduce:

- I will → **I’ll**
- we will → **we’ll**
- he will → **he’ll**
- she will → **she’ll**
- they will → **they’ll**

Esempi:

- If I have time, **I’ll** call you.
- If we leave now, **we’ll** be fine.
- If she arrives early, **she’ll** join us.

Allenati sul ritmo della frase intera, non parola per parola:

- If I have time, I’ll help you.
- If the traffic is light, we’ll get there early.
- If you don’t ask, you won’t know.

## ⚠️ Error clinic: common Italian mistakes

### Errore 1: *will* dopo *if*

❌ If I will see him, I’ll tell him.
✅ If I see him, I’ll tell him.

### Errore 2: presente al posto del risultato futuro

❌ If I study, I pass the exam.
✅ If I study, I’ll pass the exam.

Questa frase potrebbe sembrare comprensibile, ma non è la forma standard per parlare di una conseguenza futura specifica.

### Errore 3: uso meccanico della stessa struttura ovunque

A volte serve il Zero Conditional, non il First.

- If people don’t drink water, they get thirsty. ✅

Qui non stai parlando di una situazione futura singola, ma di una verità generale.

### Errore 4: confusione tra *if* e *when*

- **if** = se accade
- **when** = quando accadrà / nel momento in cui accade

Confronta:

- If he calls, I’ll tell him. → non so se chiamerà
- When he calls, I’ll tell him. → penso che chiamerà, è più probabile/certo

Questa differenza è molto utile in contesti realistici.

## 🔄 Controlled practice

### A. Complete with the correct form

1. If we ______ (leave) now, we will catch the bus.
2. If she ______ (not study), she will not pass.
3. If the weather ______ (improve), we will go out.
4. If I ______ (have) a question, I will ask.
5. If they ______ (miss) the deadline, the client will complain.
6. If you ______ (not back up) your files, you will lose them.
7. If he ______ (finish) early, he will join us.
8. If the shop ______ (be) closed, we will order online.

### B. Rewrite using the First Conditional

1. You ask me. I help you.
2. We don’t book now. We lose our seats.
3. She arrives on time. We start at nine.
4. I don’t feel better. I go to the doctor.
5. The presentation goes well. We celebrate.

### C. Choose the best option

1. If it rains tomorrow, we **stay / will stay** inside.
2. If he **will call / calls**, tell him I’m busy.
3. If they don’t hurry, they **miss / will miss** the train.
4. If I see the problem, I **fix / will fix** it.
5. If the price goes up, we **won’t buy / don’t buy** it.
