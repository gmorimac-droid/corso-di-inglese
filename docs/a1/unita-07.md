# Unità 07 – There is / There are

---

## 🎯 Obiettivo

Alla fine di questa unità sarai in grado di:

- descrivere una stanza o un luogo
- dire cosa c’è (e cosa non c’è)
- usare *there is / there are* correttamente
- parlare in modo più naturale e descrittivo

---

## 🧠 Grammar in context

### There is (singolare)

👉 per UNA cosa

- There is a book  
- There is a chair  

---

### There are (plurale)

👉 per PIÙ cose

- There are two books  
- There are three chairs  

---

### Esempi

- There is a table in the room  
- There are two windows  
- There is a computer on the desk  

---

## 🔊 Pronunciation

Forma naturale:

- There is → **There’s**  
- There are → spesso veloce: *there’re*

👉 Usa:

- There’s a book  
- There are two books  

---

## ⚠️ Errori comuni (italiani)

❌ There are a book  
✅ There is a book  

❌ There is two chairs  
✅ There are two chairs  

❌ Is a table  
✅ There is a table  

---

## 🧠 Pattern fondamentale

- There is + 1 cosa  
- There are + più cose  

👉 semplice ma fondamentale
