# Unità 04 – Everyday Objects

---

## 🎯 Obiettivo

Alla fine di questa unità sarai in grado di:

- nominare oggetti comuni
- usare *this / that / these / those*
- distinguere tra singolare e plurale
- fare domande su oggetti

---

## 🧠 Grammar in context

### This / That (singolare)

👉 Per UN oggetto

- This = questo (vicino)
- That = quello (lontano)

### Esempi

- This is a book  
- That is a car  

---

### These / Those (plurale)

👉 Per PIÙ oggetti

- These = questi  
- Those = quelli  

### Esempi

- These are books  
- Those are cars  

---

### Schema chiaro

| Vicino | Lontano |
|--------|--------|
| This (sing.) | That (sing.) |
| These (pl.) | Those (pl.) |

---

## 📚 Core vocabulary

### Oggetti comuni

- book  
- phone  
- computer  
- table  
- chair  
- bag  
- pen  
- notebook  
- door  
- window  

---

### Singolare vs plurale

- book → books  
- chair → chairs  
- pen → pens  

👉 Regola base: aggiungi **-s**

## 🎧 Listening / Reading

**Dialogue:**

A: What is this?  
B: It’s a phone.  

A: And what are these?  
B: They are books.  

A: Is that your bag?  
B: Yes, it is.

---

## 🔊 Pronunciation

Attenzione a:

- this → /ðɪs/  
- these → /ðiːz/  

👉 Suono **TH** (molto importante)

Non dire: *dis* ❌  
Ma: *this* ✅  

---

## ⚠️ Errori comuni (italiani)

❌ This are books  
✅ These are books  

❌ These is a book  
✅ This is a book  

❌ That are cars  
✅ Those are cars  

👉 Singolare e plurale devono essere coerenti
