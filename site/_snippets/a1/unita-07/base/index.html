    Base

    Modalità Base attiva. Questa versione ti aiuta a usare there is / there are per descrivere una stanza, un ufficio o un altro luogo in modo semplice. Per vedere la versione estesa usa il toggle BASE / PRO nella barra in alto.

    🎯 Obiettivo

    Alla fine di questa unità sarai in grado di:

    • dire cosa c’è o cosa non c’è in un luogo
    • usare there is con il singolare
    • usare there are con il plurale
    • descrivere una stanza con frasi semplici e corrette

    🧠 Grammar in context

    There is

    Usiamo there is per dire che c’è una sola cosa.

    • There is a table.
    • There is a bed in the room.
    • There is a lamp on the desk.

    There are

    Usiamo there are per dire che ci sono più cose.

    • There are two chairs.
    • There are three books on the table.
    • There are two windows in the room.

    Schema rapido

    Singular Plural
    There is There are

    Forme negative

    • There isn’t a TV.
    • There aren’t any plants.

    Domande utili

    • Is there a bathroom?
    • Is there a desk?
    • Are there two chairs?
    • Are there any books?

    Risposte brevi:

    • Yes, there is.
    • No, there isn’t.
    • Yes, there are.
    • No, there aren’t.

    🎧 Listening / Reading

    Text:

    There is a desk in my room.
    There is a lamp on the desk.
    There are two books next to the lamp.
    There is a chair under the window.
    There are three pictures on the wall.
    There isn’t a TV in the room.

    🗣️ Speaking

    Ripeti:

    • There is a table in the room.
    • There is a lamp on the desk.
    • There are two chairs.
    • There are three books on the shelf.
    • There isn’t a sofa.
    • Are there any windows?

    Ora completa con informazioni vere:

    • There is a ______ in my room.
    • There are ______ on my desk.
    • There isn’t a ______ in the room.
    • There are ______ windows.

    ✍️ Writing

    Scrivi 6 frasi per descrivere una stanza.

    Guida:

    1. a bed / a desk / a table
    2. one lamp
    3. two or three books
    4. one window
    5. something that is not in the room
    6. one object with on / under / next to

    Esempio:

    There is a bed in the room.
    There is a desk near the window.
    There is a lamp on the desk.
    There are two books on the desk.
    There is a chair next to the desk.
    There isn’t a TV.

    🔊 Pronunciation

    Forma contratta

    Nel parlato naturale:

    • There is → There’s

    Esempi:

    • There’s a chair.
    • There’s a lamp on the table.

    Con il plurale di solito manteniamo chiaramente:

    • There are two books.
    • There are three windows.

    ⚠️ Errori comuni

    ❌ There are a chair.
    ✅ There is a chair.

    ❌ There is two books.
    ✅ There are two books.

    ❌ Is a desk in the room.
    ✅ There is a desk in the room.

    ❌ There have two windows.
    ✅ There are two windows.

    👉 Controlla sempre se il nome è singolare o plurale.

    ✅ Review & Practice

    Completa

    1. There ___ a bed in the room.
    2. There ___ two chairs.
    3. There ___ a lamp on the desk.
    4. There ___ three books on the shelf.
    5. There ___ not a TV.
    6. ___ there any windows?

    Trasforma

    1. There is a chair. → (plurale) ______
    2. There are two books. → (singolare) ______
    3. There is a lamp. → (negativa) ______
