    Unità 08 – Places in Town

    🎯 Obiettivo

    Alla fine di questa unità sarai in grado di:

    • parlare della tua città
    • nominare luoghi comuni
    • fare domande con Is there...? / Are there...?
    • capire e partecipare a mini conversazioni reali

    🧠 Grammar in context

    Domande con There is / There are

    👉 Struttura:

    • Is there + singolare?
    • Are there + plurale?

    Esempi

    • Is there a restaurant?
    • Is there a bank?
    • Are there shops?
    • Are there parks?

    Risposte

    • Yes, there is / No, there isn’t
    • Yes, there are / No, there aren’t

    Esempi completi

    • Is there a supermarket? → Yes, there is
    • Are there restaurants? → Yes, there are

    📚 Core vocabulary

    Luoghi in città

    • restaurant
    • bar
    • café
    • supermarket
    • bank
    • school
    • hospital
    • park
    • shop
    • hotel

    Espressioni utili

    • near
    • far
    • in the city
    • in my town

    🎧 Listening / Reading

    Dialogue:

    A: Is there a restaurant near here?
    B: Yes, there is.

    A: Are there shops?
    B: Yes, there are.

    A: Is there a bank?
    B: No, there isn’t.

    🗣️ Speaking

    Ripeti:

    • Is there a restaurant?
    • Are there shops?
    • There is a park
    • There are two cafés

    👉 Parla della tua città:

    • There is a ___
    • There are ___

    👉 Fai domande:

    • Is there a ___?
    • Are there ___?

    ✍️ Writing

    Scrivi 5–7 frasi sulla tua città:

    Esempio:

    There is a supermarket.
    There is a park.
    There are restaurants.
    There is a school.
    There are shops.

    🔊 Pronunciation

    Attenzione:

    • restaurant → /ˈres.tə.rɒnt/
    • café → /kæˈfeɪ/

    👉 Non pronunciare tutte le lettere come in italiano

    ⚠️ Errori comuni (italiani)

    ❌ Is there restaurants?
    ✅ Are there restaurants?

    ❌ There is many shops
    ✅ There are many shops

    ❌ Is there a shops?
    ✅ Are there shops?

    🧠 Pattern fondamentale

    • Is there + 1 cosa?
    • Are there + più cose?

    👉 risposta breve:

    • Yes, there is
    • Yes, there are

    ✅ Review & Practice

    Completa:

    1. ___ there a bank?
    2. ___ there shops?
    3. There ___ a restaurant
    4. There ___ two parks

    🔄 Trasforma

    Trasforma in domanda:

    1. There is a bank → ______?
    2. There are shops → ______?

    🧠 Think & Speak

    Descrivi:

    👉 la tua città
    👉 un posto che conosci

    🚀 Mini task

    Simula una situazione reale:

    👉 Sei un turista

    Fai 3 domande:

    • Is there a restaurant?
    • Is there a hotel?
    • Are there shops?

    💬 Extra practice

    Rispondi:

    • Is there a park in your city?
    • Are there restaurants?
    • Is there a hospital?

    🎯 Challenge

    Aggiungi:

    • 5 luoghi diversi
    • 2 domande
    • 2 risposte

    English is useful when you can ask for what you need.
